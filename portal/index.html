Guide

Operator workflow

1

Unlock the console

Paste the server key from data/zerotier/config/file_server.key. The key is stored only in this browser.

2

Create and configure a network

Create a network, apply Easy setup, then adjust routes, assignment pools, DNS, and access control as needed.

3

Distribute client commands

Generate a temporary Linux or macOS installer. Users enter the Network ID when prompted.

4

Authorize members

Approve new devices in Members, add friendly names, and verify peer status after authorization.
