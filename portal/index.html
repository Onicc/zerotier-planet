Tutorial

Deployment and client guide

Deploy the server

  1. Run deploy.sh and choose install.
  2. Open the portal on the file server port.
  3. Copy the server key from data/zerotier/config/file_server.key.
  4. Open the controller and change the default password.

Create a network

  1. Open ztncui from the controller button.
  2. Create a network and configure the address pool.
  3. Copy the Network ID for client enrollment.
  4. Authorize new members after they run the installer.

Linux users

  1. Copy the Linux command from this portal.
  2. Run it in a terminal with sudo.
  3. Enter the Network ID when prompted.
  4. Wait for the operator to authorize the device.

macOS users

  1. Install ZeroTier One first if Homebrew is not available.
  2. Run the macOS command in Terminal.
  3. Enter the Network ID when prompted.
  4. Approve permissions requested by macOS if needed.
