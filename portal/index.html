Operator workflow

1

Create a network

Use Networks to create one or more virtual networks, then apply Easy setup for the common IPv4 case.

2

Distribute installers

Use Client delivery to generate temporary Linux or macOS commands. Links expire automatically.

3

Authorize members

New clients appear in Members. Authorize trusted devices and add clear labels for operations.

4

Operate safely

Use Settings to change or reset the console password. Back up the persistent data directory regularly.
